# Bootstrap 5 Unibit theme

## Installation
```sh
git clone https://github.com/NicolasRoehm/bootstrap-5-unibit-theme
npm i
```

Run in separated 2 terminals :
```sh
npm run dev
npm run js-watch
```

**/!\ Important**

- Unibit only works fine on Linux
